# TaskForge
Task Management website

## Description
A website to manage daily tasks efficiently.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Filter tasks by status

## Technologies Used
- HTML
- CSS
- Python
- SQL

## How to Use
1. Open the website
2. Type your task in the input box
3. Click "Add Task" button
4. Manage your tasks easily
